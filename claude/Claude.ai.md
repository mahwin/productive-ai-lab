## 클로드 특징
- 클로드는 대용량 컨텍스트 창을 보유하고 있다. 이 창은 약 20만 개 이상의 토큰을 처리할 수 있어 **방대한 컨텍스트를 기반한 작업**이 가능하다.
-  클로드와의 협력을 극대화하기 위해서는 해당 업무의 **컨텍스트를 충분히 제공**해야 한다.
- 클로드와 적절한 상호작용을 위해 대화에 **상황 설정, 과제 정의, 규칙 명시**를 포함하는 것이 좋다.
- AI를 잘 활용하기 위해서는 네 가지 핵심 역량이 필요하다.
	1. 위임 : 인간과 AI의 작업 분배를 결정하는 역량
	2. 설명 : 출력 결과, 작업 프로세스, AI 동작 및 상호작용 등을 구체적 설명하는 역량
	3. 분별력 : AI의 작업을 비판적으로 평가하는 역량
	4. 성실성 : AI를 책임감 있고 윤리적으로 사용하는 역량

## 클로드 프로젝트
- 클로드 프로젝트는 특정 작업을 수행하기 위한 환경이 구축된 **독립적인 작업 공간**이다.
- 프로젝트는 공유 가능하며, **공유를 통해 팀 내에 동일한 맥락**을 활용할 수 있다.
	- Claude for Work(팀 및 엔터프라이즈) 플랜 필요
- 컨텍스트로 부여한 **모든 정보**는 프로젝트 내에서 일어나는 **모든 작업에 반영**된다.
- RAG(Retrieval Augmented Generation)라는 기능을 통해 대용량 컨텐츠도 자동으로 처리할 수 있다.
	- RAG : 검색 보강 기술로, 대규모 콘텐츠 중 필요한 정보만을 추출해 활용하는 방식
## 클로드 스킬
- 스킬은 클로드가 특정 **작업 수행 능력**을 향상시키기 위해 동적으로 불러오는 **스크립트 및 리소스 폴더**이다.
- 스킬은 Anthropic에서 관리하는 기본 내장 스킬과 Custom 스킬로 나뉜다.
- 명시적으로 특정 스킬 사용을 요청할 수도 있지만, 일반적으로는 **Claude의 판단에 따라 적절한 스킬이 호출**된다.

## 클로드 프로젝트 vs 클로드 스킬
- 프로젝트는 컨텍스트를 저장하고, 스킬은 작업을 수행한다. 
- 프로젝트는 클로드가 특정 작업을 수행하는 데에 있어 필요한 **모든 맥락을 저장하는 독립적인 작업 공간**을 정의하기 위해 사용된다.
- 스킬은 클로드가 일관된 방식으로 반복 실행해야 하는 **워크플로우를 정의**하기 위해 사용된다.

## Connector
- 커넥터를 사용하면 클로드에게 **도구에 대한 접근 권한**을 부여할 수 있다.
- MCP(Model Context Protocol)라는 **AI용 범용 표준**을 사용하여 다양한 어플리케이션에 연결할 수 있다.
- 웹 기반 클로드를 사용하면 **클라우드 서비스에** **연결**할 수 있으며, 클로드 데스크톱을 사용하면 로컬 파일, 어플리케이션 등 **로컬 환경에 접근**할 수 있다.

## Research Mode
- Research Mode는 클로드가 웹과 연결된 **여러 플랫폼에서 수집한 정보를 종합하여 다양한 관점에서 질문을 탐구**하도록 한다.
- Research Mode는 여러 출처의 정보를 종합하고, 다양한 관점을 비교하며, 결과를 실행 가능한 통찰력으로 도출하는 등 **철저한 이해를 필요로 하는 상황을 위해 설계**됐다.
- 진행 과정
	1. 사용자의 요청을 세분화하고, 필요한 정보를 파악하며, 다양한 **관점을 조사하는 방법을 계획**
	2. 다양한 검색과 **반복적 피드백을 통해 유망한 정보를 취득**한다.
	3. 모든 정보를 종합적이고 체계적인 보고서로 **정리**한다.
	4. 정보의 **출처를 표기**한다.
- Research는 과제의 복잡성에 따라 수십 분이 걸릴 수 있기 때문에 질문을 잘 구성하는 것이 중요하다.

## Claude.ai를 사용하는 다양한 방법
- 클로드는 사용자의 작업 환경에 맞춰 설계된 여러 특수 도구에서도 사용할 수 있다. 
### Claude Code
- 터미널, IDE, 브라우저, Slack 등의 환경에서 작동하는**에이전트 기반 코딩 도구**
- **자연어**를 통해 전체 개발 워크플로우를 **처리**한다.
### Claude in Slack
- Claude는 Slack과 직접 통화되어 채널 및 스레드에서 도움을 받을 수 있다.
- @Claude를 태그하면 주변 컨텍스트를 활용하여 Claude Code 세션을 시작할 수 있다.
### Claude for Excel
- Claude는 Excel의 사이드바를 통해 스프레드시트를 분석, 이해 및 수정할 수 있다.
### Claude for Chrome
- Claude for Chrome는 크롬 확장 프로그램으로 사용자가 작업하는 내용을 관찰하고 브라우저 내에서 직접 작업을 수행할 수 있다.
